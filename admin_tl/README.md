# ComplianceOS Admin Console

## Requirements

- Node.js 20+
- npm

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React

Notes:
- This project uses the `@` path alias to reference `src/` (configured in `vite.config.ts` and `tsconfig.json`).
- Use npm as the package manager for reproducible installs (remove other lockfiles before zipping).
